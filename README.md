# Rexy Edits — Cinematic Creator Portfolio

> Professional Video & Photo Editor portfolio website for **RexyEdits**

## 🚀 Deploy to GitHub Pages

### Step 1 — Create a GitHub Repository
1. Go to [github.com/new](https://github.com/new)
2. Name it `cinematic-creator` (or anything you like)
3. Set it to **Public**
4. Click **Create repository**

### Step 2 — Upload Files
Upload these files to your repo:
- `index.html`
- `editor.png` *(your profile photo — see note below)*

### Step 3 — Enable GitHub Pages
1. Go to your repo → **Settings** → **Pages**
2. Under **Source**, select `main` branch → `/ (root)`
3. Click **Save**
4. Your site will be live at: `https://yourusername.github.io/cinematic-creator`

---

## 🖼️ Adding Your Photo

The site expects a file named `editor.png` in the root folder.

- Upload your photo and name it **`editor.png`**
- Recommended: portrait crop, min 600×800px
- If no image is provided, a cinematic placeholder is shown automatically

---

## ✏️ Customisation

All content is inside `index.html`. Search for these placeholders to replace:

| Placeholder | Replace with |
|---|---|
| `YOUR_EMAIL` | Your actual email address |
| `YOUR_DISCORD_INVITE` | Your Discord invite link |
| `YOUR_REDDIT_URL` | Your Reddit profile URL |
| `rexyedits@upi` | Your actual UPI ID |
